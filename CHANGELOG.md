# Changelog

All notable changes to this project will be documented in this file.

## [3.7.0] - 2026-07-22

### Changed
- 將「專案管理」與原本的「資料夾管理」整合為單一「專案與資料夾管理」工作區。
- 使用者選擇專案後，可直接查看專案儀表板、最近操作紀錄與資料夾清單。
- 「開啟」專案操作改為「管理」，降低介面複雜度。

### Preserved
- 建立新的專案資料夾或使用既有資料夾。
- 建立新專案後自動帶入專案資料夾 ID 到「建立位置與規則」。
- 建立前智慧檢查與失敗項目重試。
- 專案編輯、複製、封存、解除封存與刪除專案紀錄。
- 單一／批次移到垃圾桶、還原與 Drive 狀態同步。

## [3.6.0] - 2026-07-22
- 新增兩種專案資料夾模式：建立新的專案資料夾、使用既有資料夾。
- 自動建立模式會把專案資料夾 ID 帶入建立位置。

## [3.5.0] - 2026-07-22
- 建立新專案時，可在指定母資料夾中自動建立「專案名稱」資料夾。

## [3.4.0] - 2026-07-22
- 新增建立前智慧檢查。
- 新增只重試失敗項目。

## [3.3.0] - 2026-07-22
- 新增安全刪除專案功能；僅刪管理紀錄，不自動刪除 Google Drive 實體資料夾。

## [3.2.0] - 2026-07-22
- 新增專案儀表板、專案編輯、封存篩選與解除封存。

## [3.1.0] - 2026-07-22
- 新增單一／批次移到垃圾桶、還原與 Drive 狀態同步。
