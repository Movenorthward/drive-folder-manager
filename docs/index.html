Drive Folder Manager
for Education

為教育行政與學校專案設計的 Google Drive 資料夾管理工具。支援專案管理、批次建立、子資料夾範本、權限設定、智慧檢查、垃圾桶與還原。

開啟 Web App 查看 GitHub 原始碼

📁 專案與資料夾管理

以專案為核心整合資料夾、統計、操作紀錄、封存與管理。

⚡ 批次建立

支援單筆、批次貼上、CSV、手動輸入與子資料夾範本。

🔐 權限管理

可設定 Viewer / Editor，並支援同一資料夾多位使用者。

🧪 建立前檢查

先檢查母資料夾、重複名稱、Email 與預計建立數量，再正式執行。

↻ 失敗重試

大量建立時只重試失敗項目，避免成功項目被重複建立。

🗑 安全刪除

資料夾以 Google Drive 垃圾桶與還原為主；刪除專案不會自動刪除 Drive 實體資料。
